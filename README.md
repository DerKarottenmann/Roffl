Project

Project ist eine Problem-Lösungs-Website, die ähnlich wie Reddit aus Post von Usern besteht. Die Website ist rein für Projekideen und andere Ziele(z.B das Trainieren auf einen Marathon) sowie natürlich deren Umsetzung gedacht. Hierbei posten die Ersteller des Projekts, täglich, wöchentlich, etc. ihren Projektfortschritt, in einem einfach und schnellem aber informierenden Format(Steckbriefählich) , aber so, dass man die Entwicklung des Projektes über die Zeit nachvollziehen kann. Durch bestimmte Features hebt sich Project von anderen reinen Diskussionsforen wie auch Reddit ab. Project sorgt damit auch dafür, dass Ersteller zielgerichteter und produktiver sind, da sie von ihren Verfolgern motiviert werden (Social Motivation).
