Project

Roffl ist eine Problem-Lösungs-Website, die ähnlich wie Reddit aus Post von Usern besteht. Die Website ist rein für Projekideen und andere Ziele(z.B das Trainieren auf einen Marathon) sowie natürlich deren Umsetzung gedacht. Hierbei posten die Ersteller des Projekts, täglich, wöchentlich, etc. ihren Projektfortschritt, in einem einfach und schnellem aber informierenden Format(Steckbriefählich) , aber so, dass man die Entwicklung des Projektes über die Zeit nachvollziehen kann. Durch bestimmte Features hebt sich Roffl von anderen reinen Diskussionsforen wie auch Reddit ab. Roffl sorgt damit auch dafür, dass Ersteller zielgerichteter und produktiver sind, da sie von ihren Verfolgern motiviert werden (Social Motivation). 

Für die Entwicklung:

create venv: python3.12 -m venv .venv
activate venv: source .venv/bin/activate
get packages: pip install -r requirements.txt
